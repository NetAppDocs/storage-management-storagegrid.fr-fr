= Supprimer un système StorageGRID
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Supprimez un système StorageGRID si vous ne souhaitez plus le gérer depuis la NetApp Console.

.Rôle de NetApp Console requis :
Administrateur de stockagelink:https://docs.netapp.com/us-en/bluexp-setup-admin/reference-iam-predefined-roles.html["En savoir plus sur les rôles d’accès à la NetApp Console ."^]

La suppression du système de la console n’affecte pas le système StorageGRID .  Vous pouvez le rajouter à la console à tout moment.

.Étapes
. Dans le menu de navigation, sélectionnez *Stockage > Gestion*.
. Depuis la page *Systèmes*, ouvrez le système StorageGRID .
. Dans la vue standard, sélectionnez l’icône de menu et sélectionnez *Supprimer de l’espace de travail*.
. Sélectionnez *Supprimer* pour confirmer.
